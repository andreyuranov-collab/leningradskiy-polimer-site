Privacy Policy

Effective date: 15 June 2026. This policy describes data handling for the LP Comment Research Parser and related internal research pages.

1. Overview

The LP Comment Research Parser is a private internal analytics tool used to analyze public technical discussions on YouTube. It is not a public consumer service and does not provide user accounts.

2. Data collected through YouTube Data API Services

The tool may collect public YouTube data available through read-only YouTube Data API endpoints, including public video metadata, public top-level comments, public replies, timestamps, like counts, video IDs, comment IDs, author display names where returned by the API, and parent-child comment relationships.

3. Data not collected

  • No Google account passwords or credentials are collected.
  • No OAuth sign-in is required for users.
  • No private YouTube data is accessed.
  • The tool does not upload, modify, delete, moderate, or publish YouTube content.

4. Purpose of processing

Collected public data is used for offline topic analysis, clustering of recurring user problems, identifying technical issues in 3D printing discussions, product research, and market research.

5. Storage and retention

Data is stored locally or in internal research storage controlled by the operator. Reports are used internally. Data is retained only as long as reasonably necessary for research, verification, and audit purposes.

6. Sharing and sale of data

Public YouTube data collected by the tool is not sold. It is not made available through a public interface. Research summaries may be generated internally, but the tool is not designed to republish comment datasets.

7. Security

The tool uses local files, incremental writes, checkpoint/resume logs, duplicate removal, and error logs. API keys are stored outside the source code and are not published in repositories.

8. Contact

Questions about this policy may be sent to info@npolp.com.
