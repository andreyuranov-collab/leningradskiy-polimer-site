Terms of Use

Effective date: 15 June 2026. These terms apply to the private internal LP Comment Research Parser and the related evidence pages.

1. Private internal tool

The LP Comment Research Parser is a private internal research and analytics tool. It is not a public service, does not provide public accounts, and is not intended for consumer use.

2. Read-only YouTube API usage

The tool uses read-only YouTube Data API Services to collect publicly available metadata, comments, and replies from selected channels and videos. It does not upload, modify, delete, moderate, rate, or publish YouTube content.

3. No OAuth user access

The tool does not require users to sign in with their Google Account. It uses an API key for read-only public data access.

4. Internal research purpose

The tool is used for offline topic analysis, user problem clustering, technical discussion analysis, product research, and market research in the 3D printing and additive manufacturing field.

5. Compliance

The operator intends to use YouTube Data API Services in accordance with applicable YouTube API Services Terms, policies, and quota requirements.

6. No resale or public redistribution

The tool is not designed to resell YouTube data or provide a public searchable database of comments. Reports are prepared for internal analysis and decision support.

7. Contact

Questions about these terms may be sent to info@npolp.com.
