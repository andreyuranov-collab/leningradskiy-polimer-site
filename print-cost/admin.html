Вход

Важно: это тестовая клиентская авторизация для проверки интерфейса. Для настоящей защиты коммерческих данных нужна серверная авторизация через Cloudflare Access/Workers.

Как применять изменения: кнопка «Сохранить в браузере» применит данные на этом компьютере. Для обновления сайта для всех пользователей скачайте `defaults.json` и замените файл `print-cost/data/defaults.json` в GitHub-репозитории.
