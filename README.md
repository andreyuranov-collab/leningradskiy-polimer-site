# ЛЕНИНГРАДСКИЙ ПОЛИМЕР — сайт-заглушка v03

Тестовая pre-launch версия сайта.

## Позиционирование

На этой версии сайт не раскрывает производство филамента и не заявляет готовую продуктовую линейку.

Основной фокус:
- инженерные решения для 3D-печати;
- реверс-инжиниринг;
- подбор материалов;
- разработка технологических карт;
- контроль качества сырья и готовых изделий.
