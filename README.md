# ЛЕНИНГРАДСКИЙ ПОЛИМЕР — сайт компании

Тестовая версия корпоративного сайта v02, адаптированная под брендбук компании.

## Основные принципы версии v02

- инженерная точность вместо абстрактного маркетинга;
- акцент на повторяемости партий и стабильности геометрии;
- прозрачная система наименования материалов;
- цвет по RAL или Natural;
- продуктовая архитектура: базовые, модифицированные и композитные материалы;
- B2B-коммуникация через проверяемые характеристики.

- `index.html` — главная страница
