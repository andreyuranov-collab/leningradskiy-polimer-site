# LP_Website_With_PrintCost_v00.01_test

Готовая тестовая сборка сайта НПО «ЛЕНИНГРАДСКИЙ ПОЛИМЕР» с веб-калькулятором стоимости печати.

## Что изменено

1. На главную страницу добавлена кнопка **«Запустить калькулятор печати»** в первом экране.
2. В верхнее меню добавлен пункт **«Калькулятор печати»**.
3. Добавлен отдельный блок на главной странице с описанием калькулятора.
4. В сайт добавлена папка `/print-cost/` с веб-версией калькулятора.

## Как разместить через GitHub

1. Откройте репозиторий сайта: `leningradskiy-polimer-site`.
2. Нажмите **Add file → Upload files**.
3. Перетащите содержимое этой папки в окно загрузки GitHub:
   - `index.html`
   - `style.css`
   - `script.js`
   - папку `assets`
   - папку `print-cost`
   - `README_DEPLOY.md` можно не загружать, он только для инструкции.
4. Внизу страницы в поле commit message напишите:
   `Add print cost calculator to website v00.01_test`
5. Нажмите **Commit changes**.
6. Cloudflare Pages автоматически пересоберёт сайт.

## Где будет калькулятор

После обновления сайта страница калькулятора будет доступна по адресу:

`https://leningradskiy-polimer.pages.dev/print-cost/`

## Проверка после публикации

1. Откройте главную страницу сайта.
2. Убедитесь, что в первом экране появилась кнопка **«Запустить калькулятор печати»**.
3. Нажмите кнопку.
4. Проверьте, что открылась страница `/print-cost/`.
5. Загрузите тестовый G-code и проверьте расчёт.

## Важно

Это статическая тестовая версия. Расчёт выполняется в браузере пользователя. Настройки материалов, станков и коэффициентов находятся в файле:

`print-cost/data/defaults.json`

Если коммерческие коэффициенты нельзя показывать публично, следующую версию нужно делать через серверный API/Cloudflare Worker.
